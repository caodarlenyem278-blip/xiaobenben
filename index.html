⚡ 跨设备同步

手机和电脑使用相同的 Supabase URL、Anon Key 和令牌即可实时同步

⚪ 未连接

📖 首次使用设置步骤

1. 打开 supabase.com，注册账号，创建项目

2. 在项目 Settings → API 中获取 Project URL 和 anon/public key

3. 在项目 SQL Editor 中运行下方 SQL 创建数据表

4. 在本页面填入上述信息和你自定义的令牌，点击「连接」

5. 手机和电脑都用相同令牌连接后，数据自动实时同步

所需 SQL（在 Supabase SQL Editor 中运行）：

CREATE TABLE sync_rooms (
  token TEXT PRIMARY KEY,
  data JSONB DEFAULT '{}'::jsonb,
  updated_at TIMESTAMPTZ DEFAULT NOW()
);
ALTER TABLE sync_rooms ENABLE ROW LEVEL SECURITY;
CREATE POLICY anon_access ON sync_rooms
  FOR ALL TO anon USING (true) WITH CHECK (true);
